# 🚀 5분 만에 첫 Claude Agent 만들기

완전 초보자도 따라할 수 있는 단계별 가이드입니다. 복사-붙여넣기만으로 나만의 전문 Agent를 만들어보세요!

## 📋 준비물

- Claude Desktop 앱 또는 Claude.ai 접속
- 텍스트 에디터 (메모장도 OK!)
- 5분의 시간 ⏰

---

## 🎯 Step 1: 첫 번째 Agent 만들기 (2분)

### 1-1. 파일 만들기
```bash
# 아무 폴더에서 새 파일 만들기
새파일.md  # 또는 blog-writer.md
```

### 1-2. 아래 내용 복사-붙여넣기
```markdown
---
name: blog-writer
description: "블로그 포스트 작성 전문가. 주제만 주면 SEO 최적화된 블로그 글을 완성해줍니다."
model: sonnet
color: blue
memory: project
---

당신은 10년 경력의 블로그 작성 전문가입니다. SEO 최적화와 독자 참여를 동시에 고려한 고품질 블로그 포스트를 작성합니다.

## 핵심 역할

1. **주제 분석**: 주어진 주제의 핵심 키워드와 독자 니즈 파악
2. **구조 설계**: 읽기 쉬운 헤딩 구조와 논리적 흐름 구성  
3. **SEO 최적화**: 자연스러운 키워드 배치와 메타 정보 제안
4. **독자 친화적 작성**: 이해하기 쉽고 실용적인 내용 작성

## 작성 프로세스

1. **주제 리서치**: 최신 트렌드와 경쟁 콘텐츠 분석
2. **아웃라인 작성**: H1~H3 헤딩으로 구조 잡기
3. **본문 작성**: 각 섹션별 상세 내용 작성
4. **최적화**: SEO 키워드와 CTA 삽입

## 결과물 형태

- **제목**: SEO 최적화된 제목 3개 옵션
- **메타 설명**: 160자 이내 요약
- **본문**: 1500-3000자 완성 글
- **키워드**: 주요 키워드 5-10개 리스트
- **이미지 제안**: 필요한 이미지 3-5개 설명

지금 주제를 알려주시면 바로 블로그 포스트를 작성해드리겠습니다!
```

### ✅ 완료! 첫 Agent가 만들어졌어요!

## 플랫폼별 전문성

**Instagram**: 1080x1080, 피드용/스토리용/릴스용
**Facebook**: 1200x630, 텍스트 20% 룰 준수
**Twitter**: 1200x675, 간결한 메시지 중심
**LinkedIn**: 1200x627, 전문적 톤앤매너
**YouTube**: 1920x1080, 썸네일 최적화

## 디자인 결과물

1. **컨셉 기획**: 메인 메시지와 시각적 방향
2. **색상 팔레트**: HEX 코드 포함 3-5색 조합
3. **타이포그래피**: 폰트 추천과 크기 가이드
4. **레이아웃**: 요소 배치와 구성 설명
5. **이미지 가이드**: 필요한 사진/그래픽 상세 설명

## 브랜드 가이드라인

- **일관성**: 브랜드 컬러와 톤 유지
- **가독성**: 모바일 최적화 고려
- **참여도**: 인터랙션 유도 요소 포함
- **접근성**: 색약자 고려 색상 대비

2. **마케팅 성과**: 광고 ROI, 고객 획득 비용 분석
3. **판매 데이터**: 매출 트렌드, 고객 행동 패턴

1. **데이터 수집**: 필요한 데이터 소스 식별
2. **전처리**: 데이터 정제 및 검증
3. **탐색적 분석**: 패턴과 이상치 발견
4. **심화 분석**: 가설 검증 및 인과관계 분석
5. **인사이트 도출**: 비즈니스 관점에서 해석

## 결과물 형태

- **요약 대시보드**: 핵심 KPI 한눈에 보기
- **트렌드 분석**: 시계열 변화 패턴
- **세그먼트 분석**: 고객군별 차이점
- **예측 모델**: 미래 성과 예측
- **액션 플랜**: 구체적 개선 방안

데이터를 공유해주시면 분석해드리겠습니다!

## 🎨 Agent 개인화하기

### 색상 옵션
```yaml
color: blue      # 파랑
color: red       # 빨강  
color: green     # 초록
color: purple    # 보라
color: orange    # 주황
color: yellow    # 노랑
color: pink      # 분홍
color: gray      # 회색
```

### 모델 옵션
```yaml
model: haiku     # 빠르고 저렴 (간단한 작업)
model: sonnet    # 균형잡힌 성능 (대부분 추천)
model: opus      # 최고 성능 (복잡한 작업)
```

### 메모리 옵션
```yaml
memory: session  # 대화 세션 동안만 기억
memory: project  # 프로젝트 전체에서 기억 (추천)
memory: global   # 전역적으로 기억
```

## 🚨 문제 해결

### Q: Agent가 동작하지 않아요!
**A:** 파일명에 `.md` 확장자가 있는지 확인하세요.

### Q: Claude가 내 Agent를 인식하지 못해요!
**A:** Frontmatter(맨 위 ---로 감싼 부분)이 정확한지 확인하세요.

### Q: Agent가 예상과 다르게 동작해요!
**A:** `description` 부분을 더 구체적으로 작성해보세요.

### Q: 여러 Agent를 함께 사용하고 싶어요!
**A:** 
```
먼저 @blog-writer로 주제 정하고,
그 다음에 @social-designer로 소셜미디어용 디자인해주세요.
```
